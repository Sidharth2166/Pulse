# Pulse - Daily Summary Bot

An automated Python bot that fetches:

- Live Weather (wttr.in API)
- Daily Motivational Quote (ZenQuotes API)

The bot generates a daily summary and runs automatically every day using GitHub Actions.

## Technologies Used
- Python
- Requests
- GitHub Actions
- APIs

## Features
- Automated scheduling
- Error handling
- Downloadable summary file
- Live API integration
